## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„

бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ‘бҹҮбһҹбҹҶбһҺбһ¶бһҖбҹӢбһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһӮбһ¶бһҡбһ…бҹ’бһҡбһҫбһ“ бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһҖбһ¶бһҡбһҖбһҖбҹӢбһ”бһ“бҹ’бһ‘бһ”бҹӢ бһҖбһ¶бһҡбһ…бһјбһӣ/бһ…бҹҒбһүбһ—бҹ’бһүбҹҖбһң бһңбһ·бһҖбҹ’бһҖбһҷбһ”бһҸбҹ’бһҡ бһ—бҹ„бһҮбһ“бһёбһҷбһҠбҹ’бһӢбһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбҹ” бһ”бһ»бһӮбҹ’бһӮбһӣбһ·бһҖбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҸбһ¶бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖ (browser) бһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһ—бҹ’бһҮбһ¶бһ”бҹӢ Wi-Fiбҹ”

## бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„

### бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ

- Node.js v18+
- MySQL 8.0+
- npm

### бһҮбҹҶбһ бһ¶бһ“

```bash
# 1. Clone
git clone <repo-url>
cd HappyStaySYS

# 2. Install dependencies
npm install
cd client && npm install && cd ..

# 3. Configure
cp .env.example .env
# Edit .env вҖ” set DB_PASSWORD and JWT_SECRET

# 4. Database
npm run migrate
npm run seed

# 5. Start
npm run dev
```

### Environment Variables (.env)

```env
DB_HOST=localhost
DB_PORT=3306
DB_NAME=guesthouse_db
DB_USER=root
DB_PASSWORD=

APP_PORT=3000
JWT_SECRET=your_secret_key
SESSION_EXPIRES=8h

GUESTHOUSE_NAME=HappyStay Guesthouse
CURRENCY=USD
TAX_RATE=0
DEFAULT_CHECKIN_TIME=14:00
DEFAULT_CHECKOUT_TIME=12:00
```

### Default Login

- Username: `admin`
- Password: `admin123`

### Production (PM2)

```bash
npm run build
npm install -g pm2
pm2 start npm --name "happystay" -- start
pm2 save && pm2 startup
```

Access: `http://192.168.x.x:3000`

## Database Tables

| Table | Description |
|---|---|
| `buildings` | бһўбһӮбһ¶бһҡ (name, code) |
| `floors` | бһҮбһ¶бһ“бҹӢ per building |
| `rooms` | бһ”бһ“бҹ’бһ‘бһ”бҹӢ with fan/aircon prices, discount prices |
| `room_types` | Single, Double, Twin, Suite, Family |
| `guests` | бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ—бҹ’бһүбҹҖбһң (name, ID, phone, nationality) |
| `reservations` | бһҖбһ¶бһҡбһҖбһҖбҹӢ with cooling_type, price_type, custom_price |
| `reservation_guests` | Group booking (multiple guests per room) |
| `checkins` | Actual arrival/departure records |
| `invoices` | Invoice header per booking |
| `invoice_items` | Line items: room, food, extras |
| `payments` | Payment records (USD/KHR) |
| `menu_items` | Restaurant menu (food, drinks, snacks) |
| `orders` | Restaurant order header |
| `order_items` | Food items per order |
| `staff` | Staff accounts and roles |
| `activity_log` | Action audit trail |
| `settings` | System configuration (exchange rate, etc.) |

## Security

- JWT authentication on all API routes
- Passwords hashed with bcrypt (10 rounds)
- Role-based access control per endpoint
- In-app confirmation modals for all destructive actions
- Activity log for all sensitive operations
